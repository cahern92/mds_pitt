Hierarchical Clustering

Hiearchical clutering still bases SIMILARITY on DISTANCES.

But, we do not need to specify the num of clusters upfront!

Instead, as the name states, clusters within clusters are formed!!

Hiearchical clustering will in effect try out ALL possible number of clusters for us!!!

And it provides a useful VISUALIZATION to interpret the clustering results!!

We will use sklearn quite often in this course, but for hiearchical clsutering, we will use functions from SCIPY.

Import the functions we need from SCIPY.

In[40]:
from scipy.cluster import hierarchy

Hierarchical clustering still calculates distances. Therefore, it is critical to use STANDARDIZED variables just like KMeans!!!!

However, there is another kind of distance that needs to be calculated for hierarchical computing.

We will discuss that other kind of distance after examining some results.

This other kind of distance is referred to as the LINKAGE.

Linkages:

  • COMPLETE: MAXIMUM distance between two clusters
  • SINGLE: MINIMUM distance between two clusters
  • AVERAGE: AVERAGE distance between two clusters
  • CENTROID: Calculates the center of the clusters and then calculates the distance between two clusters

COMPLETE LINKAGE

In[41]:
hclust_complete = hierarchy.complete(Xwine)
In[42]:
type(hclust_complete)
Out[42]:
numpy.ndarray

Visualize the results witha specialized fig called a DENDROGRAM.

This presents the clustering results in a TREE LIKE structure.

In[44]:
fig = plt.figure(figsize=(12, 6))

dn = hierarchy.dendrogram(hclust_complete)

plt.show()

Bottom is called the NODES or LEAVES. The top lines are the ROOTS.

The leaves represent each observation or row.

The roots represents combining all rows into one cluster.

Agglomerative approach means that we start with each individual obs, cluster the ones close to it until everything is clustered. (Divisive is the opposite)

This is shown as the top line being one cluster!

Y-axis is a measure of dissimilarity.

In[45]:
fig = plt.figure(figsize=(12, 6))

dn = hierarchy.dendrogram(hclust_complete, no_labels=True)

plt.show()
