Overview

Matplotlib has 2 main styles of programming.

The first or classic is the STATEFUL style which is based on MATLAB.

The second approach is the OBJECT ORIENTED STYLE APPROACH.

Stateful or MATLAB style

  • Step 1: Initiate the figure
  • Step 2: Specify the subplot that contains the data
  • Step 3: Populate the data within the plot (plot the data)
  • Step 4: Show the figure

Lets being with step 1 and work through each step one at a time.

Object Oriented Style

Many aspets of the object oriented style figures are the same as the STATEFUL figures. But the way we access or apply functions to them are different!

  • Step 1: Define the figure and axes objects - create variables!
  • Step 2: Plot the data within the axes using methods associated with the axes objects!
  • Step 3: Modify the axes object attributes if needed
  • Step 4: Modify the figure object attributes if needed
  • Step 5: Show the figure

The motivation for the object oriented approach is that the STATEFUL approach works with the HIDDEN OBJECTS!!!

Define variables or objets for the FIGURE and AXES that we are modifying!!

The common names for these objects are fig and ax.
