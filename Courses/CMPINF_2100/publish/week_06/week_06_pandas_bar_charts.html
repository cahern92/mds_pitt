Creating bar charts with Pandas¶

This week we have seen how to use matplotlib to organize our plots. We saw the diff between the FIGURE and AXES objects. But I will very rarely create plots in matplotlib directly. Instead statistical plots such as the BAR CHART will be created using Pandas or Seaborn. This notebook shows how to use Pandas.

Lets now visualize the counts with a BAR CHART!!

Although matplotlib will manage the figure..we will NOT call a matplotlib function to generate the plot!!

Lets use Pandas plotting methods to make the bar chart!

We must still being by DEFINING the object oriented style figure and axes objects!!

In [17]:
fig, ax = plt.subplots()

gap_df.continent.value_counts().plot(kind="bar", ax=ax)

plt.show()
In [22]:

gap_df.continent.value_counts().plot(kind="bar", ax=ax)

plt.show()
